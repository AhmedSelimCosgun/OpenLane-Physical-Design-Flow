# ASIC Physical Design Portfolio (Sky130 PDK)

Welcome to my VLSI Physical Design portfolio. This repository contains my projects demonstrating the complete RTL-to-GDSII pipeline using the open-source **SkyWater 130nm PDK**, **OpenLane**, and **OpenROAD**.

## 📁 [01_Automated_Pushbutton_Flow](./01_Automated_Pushbutton_Flow)
**The "Efficiency" Approach.** An introductory run utilizing OpenLane's automated macro-compiler mode. Contains the baseline configuration and source code.

## 📁 [02_Interactive_Custom_Flow](./02_Interactive_Custom_Flow)
**The "Deep Dive" Approach.** Executed entirely in Interactive Mode. It demonstrates manual intervention and granular control over the physics of the chip, including custom floorplanning, manual I/O pin placement, custom power rings, and Clock Tree Synthesis analysis.
